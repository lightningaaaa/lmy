4、韦诺秘籍？？
http://bbs.xiawai.com/thread21416p1v1
使用方法：
进入游戏 按 ：输入 debug 回车确认开启秘籍模式。
然后按 ：输入以下秘籍可得到对应效果：
秘籍	作用
unit hitpoints=#	设置选中单位的血量，#是数量
unit moves=#	设置选中单位移动步数 #为（1-99）
unit experience=#	设置选中单位的经验值
create [#]	制造单位
gold #	增加金钱 #为（0-999999）
unit level=#	设置选中单位等级
